# Douravita CLI

CLI wrappers for Infisical and Railway, built with the [cli-anything](https://github.com/Ianfr13/cli-anything) methodology.

## CLIs

### `cli-anything-infisical`
CLI wrapper for [Infisical](https://infisical.com) secrets manager.

```bash
cd infisical
pip install -e .
```

### `cli-anything-railway`
CLI wrapper for [Railway](https://railway.app) deployment platform.

```bash
cd railway
pip install -e .
```
